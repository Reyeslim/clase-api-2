Guía Rápida de Sintaxis

.then() / .catch()

  • Encadenar con .then()
  • Errores con .catch()
  • return para pasar datos
  • Sintaxis tradicional (ES2015)

async / await

  • Función debe ser async
  • await espera la promesa
  • try/catch para errores
  • Sintaxis moderna (ES2017)
